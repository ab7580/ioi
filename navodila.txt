Prerequisite: Python (preverjeno na 3.8, ostale verzije bi tudi morale delati)

1. cd-jaj git folder (pazi, da nimaš presledkov in šumnikov v pathu)
2. python -m venv ioienv
3. ioienv\Scripts\activate
4. pip install -r requirements.txt (to poženeš tudi vsakič, ko kej spremenimo od packagov)
5. python manage.py runserver 
6. V brskalniku odpri http://127.0.0.1:8000/
